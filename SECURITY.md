# Security Policy

WorkTraceAgent 会读取本机 Coding Agent 会话，因此路径脱敏、凭据排除、只读采集和外部请求边界都属于安全关键功能。

## 报告漏洞

请优先使用 GitHub 仓库的 **Security → Report a vulnerability** 私密报告入口。如果该入口不可用，请创建一个不包含漏洞细节、凭据、私有路径或真实会话内容的最小 issue，请维护者提供私密联系方式。

报告中建议包含：

- 受影响版本与操作系统；
- 最小复现步骤和预期/实际行为；
- 可能泄露或被修改的数据类别；
- 已知缓解方式。

请勿在公开 issue 中粘贴真实用户会话、访问令牌、私钥、Cookie、完整本机路径或尚未修复的利用代码。

## 支持范围

最新发布版本和 `main` 分支接受安全修复。旧版本不保证单独维护；修复通常进入下一个补丁或次版本。若问题可能导致凭据、私有会话、OKR、工作画像或文件路径离开本机，请将其按高优先级报告。
